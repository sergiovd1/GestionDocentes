Formato CSV Requerido

El archivo debe usar punto y coma (;) como separador. La primera fila es para encabezados.

Siglas;Nombre;Email;Departamento_id;Tipo;Antiguedad;Nota
JPER;Juan Perez;juan@email.com;1;Interino;2020-09-01;8.5
AMAR;Ana Maria;ana@email.com;2;Carrera;2018-09-01;9.2

• Siglas: Código único (usuario para login).

• Tipo: Carrera, Interino, En prácticas.

• Antigüedad: Formato AAAA-MM-DD.

• Nota: Usar punto (.) para decimales.
